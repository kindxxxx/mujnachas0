# Промпт для Claude Code: Создание сайта-лендинга «Профессиональные строительные и отделочные бригады»

**Контекст задачи:** Необходимо создать современный, стильный и профессиональный одностраничный сайт (лендинг) для команды квалифицированных специалистов по ремонту, строительству и бытовым техническим задачам. Уйти от стереотипного образа «муж на час» в сторону позиционирования надежной строительно-монтажной компании или сервиса полного цикла. Сайт должен быть полностью готов к деплою на **GitHub Pages** и включать интерактивную форму заявки с отправкой данных в **Telegram**.

---

## Техническое задание для Claude

Создай полностью готовый, современный и адаптивный одностраничный сайт (`index.html`) с использованием Tailwind CSS (через CDN) и Vanilla JavaScript. Дизайн должен выглядеть дорого, авторитетно и технологично (строгие цвета, глубокий графит/темно-синий или чистый благородный минимализм, акцентные кнопки, карточки с легкими тенями, высокая конверсия).

### 1. Позиционирование и Шапка сайта (Hero-блок)
* **Название бренда / Заголовок:** Профессиональные инженерно-строительные и ремонтные решения «под ключ».
* **Подзаголовок:** Комплексный ремонт, отделочные работы, электромонтаж, сантехника и металлоконструкции любой сложности в Алматы. Работаем по договору, со своими инструментами, быстро и с гарантией качества.
* **Кнопки CTA (Call to Action):** «Оставить заявку», «Оценить стоимость по фото» (с плавным скроллом к форме).

### 2. Структура каталога услуг (Сетка карточек с иконками)
Оформи услуги в виде удобных интерактивных карточек:
1. **Инженерные и электромонтажные работы:** Замена розеток и выключателей, монтаж светильников и люстр, частичная прокладка проводки, устранение неполадок.
2. **Сантехнические работы:** Замена смесителей, профессиональный монтаж раковин, инсталляций и унитазов, оперативное устранение протечек.
3. **Отделочные работы и ремонт:** Малярные работы (шпаклёвка, покраска), мелкосрочный косметический ремонт, качественная герметизация и гидроизоляция.
4. **Сборочные и монтажные работы:** Установка стеллажей, зеркал, карнизов, комплексная сборка корпусной мебели любой сложности, регулировка и устранение перекосов.
5. **Изготовление металлоконструкций:** Профессиональная сварка, сборка металлических каркасов, усиление несущих элементов и конструкций.
6. **Специальные и сопутствующие задачи:** Профессиональный демонтаж, комплексные решения под ключ, нестандартные технические задачи.

### 3. Преимущества компании
* Профессиональная команда профильных специалистов со стажем.
* Профессиональный электроинструмент и спецоборудование для любых задач.
* Строгое соблюдение чистоты на объекте (аккуратность, уборка после работ).
* Оперативный выезд на объекты, возможность срочного выполнения заказов.
* Предварительная оценка стоимости по фотографиям (отправка через форму).

### 4. Интеграция с Telegram (Обратная связь)
* Создай стильную форму обратной связи с полями: **Имя**, **Номер телефона**, **Описание задачи / пожелания**, а также возможность указать ссылку на фото (или прикрепить описание).
* Реализуй отправку данных из формы напрямую в Telegram-чат с использованием Telegram Bot API (`fetch` запрос на `https://api.telegram.org/bot<BOT_TOKEN>/sendMessage`).
* В самом верху скрипта оставь понятные константы:
  ```javascript
  const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE';
  const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID_HERE';
  ```
* Добавь клиентскую валидацию полей и красивые алерты/уведомления для пользователя об успешной отправке заявки или ошибке сети.

### 5. Дополнительные требования к файлам
* Выдай код в виде чистого, хорошо структурированного единого файла `index.html` (или дай структуру для репозитория GitHub, включая базовый `README.md` с инструкцией по деплою на GitHub Pages).
* Обеспечь безупречную адаптивность под мобильные устройства, планшеты и десктопы.
