# ПРОФ-МОНТАЖ — Сайт-лендинг строительно-монтажных услуг

Современный, премиальный и полностью адаптивный одностраничный сайт (лендинг) для команды специалистов по ремонту, строительству и бытовым техническим задачам в Алматы.

Сайт построен на **Tailwind CSS** (через CDN), **Lucide Icons** и ванильном **JavaScript**. Включает интерактивную форму с валидацией номера телефона и интеграцией с **Telegram Bot API** для мгновенного получения заявок.

---

## 🛠 Настройка Telegram-бота для получения заявок

Чтобы сообщения из формы обратной связи приходили напрямую вам в Telegram, выполните следующие шаги:

### 1. Создание Telegram-бота
1. Откройте Telegram и найдите официального бота **[@BotFather](https://t.me/BotFather)**.
2. Отправьте ему команду `/newbot`.
3. Задайте имя бота (например, `Мой Монтаж Сайт`).
4. Задайте уникальный юзернейм бота, заканчивающийся на `bot` (например, `prof_montazh_lead_bot`).
5. **Скопируйте полученный токен** (например, `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`). Это ваш `TELEGRAM_BOT_TOKEN`.

### 2. Получение вашего Chat ID
Вы можете получать заявки в личные сообщения или в групповой чат:

#### Получение в личные сообщения:
1. Найдите бота **[@userinfobot](https://t.me/userinfobot)** в Telegram и отправьте ему любое сообщение.
2. Бот пришлет вам сообщение, содержащее ваш **Id** (число, например, `987654321`). Это ваш `TELEGRAM_CHAT_ID`.
3. Откройте вашего созданного бота и нажмите кнопку **Запустить (Start)** (это обязательно, чтобы бот мог отправлять вам сообщения).

#### Получение в групповой чат (для команды):
1. Создайте группу в Telegram и добавьте туда вашего созданного бота.
2. Сделайте бота администратором группы.
3. Добавьте в группу бота **[@RawDataBot](https://t.me/RawDataBot)**. Он выведет JSON с информацией о группе.
4. Найдите значение `"id"` в объекте `"chat"` (оно будет отрицательным числом, например, `-1001234567890`). Это ваш `TELEGRAM_CHAT_ID`.
5. Удалите `@RawDataBot` из группы.

### 3. Интеграция токенов в код
Откройте файл `index.html` на строках 480-482 и замените значения констант на ваши данные:

```javascript
// ----------------------------------------------------
// TELEGRAM BOT API CONFIGURATION
// ----------------------------------------------------
const TELEGRAM_BOT_TOKEN = 'СЮДА_ВСТАВЬТЕ_ВАШ_ТОКЕН';
const TELEGRAM_CHAT_ID = 'СЮДА_ВСТАВЬТЕ_ВАШ_CHAT_ID';
// ----------------------------------------------------
```

## 🚀 Деплой на GitHub Pages

Сайт полностью оптимизирован для бесплатного хостинга на GitHub Pages.

### Шаг 1: Создание репозитория
1. Зарегистрируйтесь или авторизуйтесь на [GitHub](https://github.com).
2. Создайте новый публичный репозиторий с названием (например, `prof-montazh-landing`).

### Шаг 2: Загрузка файлов
Инициализируйте репозиторий локально и отправьте файлы в GitHub:

```bash
git init
git add .
git commit -m "Initial commit with premium landing page"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/prof-montazh-landing.git
git push -u origin main
```

*(Или просто загрузите файлы `index.html`, `hero.png` и `README.md` вручную через веб-интерфейс GitHub).*

### Шаг 3: Включение GitHub Pages
1. Перейдите в настройки вашего репозитория (**Settings**) на GitHub.
2. В левом боковом меню выберите раздел **Pages**.
3. В секции **Build and deployment** выберите источник **Deploy from a branch**.
4. В поле **Branch** выберите ветку `main` (или `master`) и папку `/root`, затем нажмите **Save**.
5. Через 1-2 минуты страница будет опубликована по адресу: `https://ВАШ_ЛОГИН.github.io/prof-montazh-landing/`.

## ✨ Особенности реализации
- **Премиальный дизайн**: Темная цветовая гамма (глубокий графит, темно-синий) с бирюзово-изумрудными акцентами и эффектом матового стекла (Glassmorphism).
- **Высокая конверсия**: Удобный плавный скроллинг по клику на карточки услуг с автовыбором категории в форме заявки.
- **Интерактивность**:
  - Анимированные аккордеон-ответы на популярные вопросы (FAQ).
  - Красивые всплывающие уведомления (Toast Alerts) для информирования о статусе отправки формы.
  - Маска для ввода номера телефона казахстанских операторов (`+7 (7XX) XXX-XX-XX`).
